"Upload a list" - select a json file from your PC. If the format is not right nothing will happen.
"Upload a list from clipboard": paste the copied text in the newly generated textarea. You can also use this to add a bunch of new elements to an already existing list.
"Add new element": add 1 new element at the end of the list. You can open and edit an element by clicking on the name. A dropdown box with a preview and edit box will pop up. You can edit the values in the edit box and they will automatially apply.
"X": Delete button used to remove single element.
"Save": download a new materials.json from your list.
